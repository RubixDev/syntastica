Expand description

A non-exhaustive list of text file types.

The type derives the following traits for convenience. For (de)serialization to/from strings, lowercase casing is used unless otherwise specified in the variants docs.

  • [strum::Display]: Display formatting and a .to_string() method
  • [strum::AsRefStr]: AsRef<str> impl for conversion into &str
  • [strum::IntoStaticStr]: From<FileType> impl for conversion into &'static str
  • [strum::EnumString]: FromStr impl for turning strings into the corresponding variant
  • [strum::EnumVariantNames]: an associated VARIANTS constant containing the string names of all variants (requires [strum::VariantNames] to be in scope)
  • Clone, Copy
  • Debug
  • Hash
  • PartialEq, Eq
  • Default: the default is FileType::Text
  • serde serde::Serialize: serialize into a string
  • serde serde::Deserialize: deserialize from a string
